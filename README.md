# AWS toolkit

Command line utilities:
```
ssm-session
scale-in-ecs
ses-suppression-list
ecr-gitconfig
```

## Usase

Each command has `--help` section
```bash
ssm-session --help
scale-in-ecs --help
ses-suppression-list --help
ecr-gitconfig --help
```

## Installation 

Linux:
```bash
git clone https://github.com/meslab/aws-toolkit.git
cd aws-toolkit/
make install
```
